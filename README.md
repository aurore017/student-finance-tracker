GlowBudget Journal is a simple student finance tracker that helps you record spending, search records quickly, and see useful totals.

---

## What you can do

✔ Add spending records  
✔ Edit and delete records  
✔ Search using keywords or regex patterns  
✔ Sort by date, description, or amount  
✔ View total spending and category trends  
✔ See a 7-day spending chart  
✔ Set a monthly spending cap  
✔ Switch between light and dark mode  
✔ Choose display currency (RWF, USD, EUR)  
✔ Import and export data (JSON)  
✔ Data saves automatically  

---

## Dashboard shows

- Total records  
- Total spent  
- Top category  
- Monthly cap status  
- Last 7 days spending  

---

## Search examples

Try typing:

(coffee|tea)  
→ finds drink purchases  

\.\d{2}\b  
→ finds amounts with cents  

\b(\w+)\s+\1\b  
→ finds repeated words (example: coffee coffee)

## Keyboard tips

- Press **Tab** to move between fields  
- Press **Enter** to activate buttons  
- Use the **Skip to content** link for quick navigation

## Dark Mode

Enable dark mode in **Settings → Dark Mode**  
Your preference is saved automatically.

---

## Import / Export Data

### Export
Settings → Export JSON

### Import
Settings → Import JSON → select file

The app checks the file before loading.
